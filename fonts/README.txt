CUSTOM FONTS
=============

Place your custom .ttf or .otf font files in this folder to use them in the overlay.

How to use:
1. Download a .ttf or .otf font file
2. Copy it to this folder (G:\dnd\maphelper\fonts\)
3. Restart the application

The font will automatically be loaded and used for all text in the overlay and dialogs.

Example:
  fonts/
    MyCustomFont.ttf  <- Your font file here

Recommended fonts:
- Roboto
- Source Code Pro
- Fira Code
- JetBrains Mono

Note: Only the first font file found will be loaded.
